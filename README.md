## Technologies
- HTML5
- CSS3
- JavaScript
- React
- Redux
- Sequelize
- Express

### To install packages: npm install

### `npm run start:dev`
Runs the app in the development mode.

### `npm run seed`
To seed the database

Open http://localhost:8080 to view it in your browser.
